## 📁 Project Structure

```bash
src/
├── components/        # Reusable UI components (DataHero, ProjectAnalytics, etc.)
├── layout/            # Navbar.jsx (with dynamic badge) and Footer.jsx
├── context/           # AppContext.jsx (Global Context Provider)
├── reducer/           # appReducer.js (Action logic and initial state)
├── pages/             # Home, Projects, Research, Skills, Contact, SavedAnalytics
├── App.jsx            # Main routing configuration
├── App.css            # Custom styling and Teal/Dark Slate theme variables
├── index.css          # Base CSS reset
```
## 📦 Installation & Setup
1. Clone the Repository
    for example:
 ```bash
git clone https://github.com/charu-sood/data-analytics-portfolio.git
cd data-analytics-portfolio
```

2. Install Dependencies
```bash
npm install @mui/material @emotion/react @emotion/styled @mui/lab @mui/icons-material bootstrap
```
3. Run the Development Server
```bash
npm run dev
```
